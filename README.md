# vim-asciify
Text to ascii generator Vim plugin.

## Usage
Copy the asciify dir to ~/.vim/plugin (create directory if absent)

![example](https://i.imgur.com/AlY1uja.gif)

It is possible to use a different comment delimiter (default is #).

For quotes:
```
:Asciify \"
```
And slashes:
```
:Asciify //
```
